. => tout caractère sauf \n \r
\d => équivalent à 0-9
\D => inverse \d ou ^0-9
\w => alphanumérique
\W => opposé
\s => caractère d'espaçement
\S => opposé
\u{hhh} => valable si u avant hhh
\ => traiter littéralement  /a\*/ correspond à "a*".
[a] => correspond a a
[a-c] => correspond a a to c
[a-] =>] coresspond au a et -
[^xyz] => complément xyz
x|y => soi x soi y
^ => début entrée 
$ => fin entrée
x* => répété 0 ou plusieurs fois
x+ => répété 1 ou plusieurs fois
x? => répété 0 ou 1 fois
x{n} => nb n occurence de x
x{n,} => nb min d'occurence de x
